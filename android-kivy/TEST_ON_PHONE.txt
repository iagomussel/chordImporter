📱 METHOD 2: WSL2 ON WINDOWS (Setup once, use forever)

  PowerShell (as Admin):
  ┌────────────────────────────────────────┐
  │ wsl --install                          │
  │ (restart computer)                     │
  └────────────────────────────────────────┘

  After restart, Ubuntu terminal:
  ┌────────────────────────────────────────┐
  │ cd /mnt/f/projetos/chordImporter/      │
  │     android-kivy                       │
  │ sudo apt update                        │
  │ sudo apt install -y python3-pip        │
  │     openjdk-11-jdk                     │
  │ pip3 install buildozer cython          │
  │ buildozer android debug                │
  └────────────────────────────────────────┘

  First build: 30-60 minutes (downloads Android SDK)
  Next builds: 2-5 minutes ⚡

  APK location (in Windows):
  F:\projetos\chordImporter\android-kivy\bin\chordimporter-*-debug.apk

  Transfer to phone → Install!

📲 INSTALLING ON ANDROID PHONE

  1. Transfer APK to phone (email, USB, cloud)
  2. Open file manager app
  3. Tap the .apk file
  4. If blocked: Tap Settings → Allow from this source
  5. Tap Install
  6. Open app
  7. Tap Allow when asked for microphone permission

  Done! 🎸
