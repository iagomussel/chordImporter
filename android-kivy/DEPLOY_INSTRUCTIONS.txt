  🚀 DEPLOY TO YOUR ANDROID PHONE - 3 SIMPLE STEPS
════════════════════════════════════════════════════════════════

✅ GOOD NEWS: You have WSL installed!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📱 STEP 1: PREPARE YOUR PHONE (2 minutes)

1. Enable Developer Mode:
   ┌────────────────────────────────────────────┐
   │ Settings                                   │
   │  └─ About Phone                            │
   │      └─ Tap "Build Number" 7 TIMES         │
   └────────────────────────────────────────────┘
   You'll see: "You are now a developer!"

2. Enable USB Debugging:
   ┌────────────────────────────────────────────┐
   │ Settings                                   │
   │  └─ System (or Developer Options)          │
   │      └─ Developer Options                  │
   │          └─ USB Debugging → ON             │
   └────────────────────────────────────────────┘

3. Connect Phone to PC:
   - Use USB cable
   - On phone, select "File Transfer" mode
   - Allow USB debugging (popup on phone)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 STEP 2: OPEN WSL AND NAVIGATE (30 seconds)

Option A - From Windows Explorer:
  1. Open: F:\projetos\chordImporter\android-kivy
  2. Right-click in folder
  3. Select "Open Linux shell here"

Option B - Manually:
  1. Press: Win + R
  2. Type: wsl
  3. Press: Enter
  4. Run: cd /mnt/f/projetos/chordImporter/android-kivy

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🏗️ STEP 3: BUILD & DEPLOY (15-30 minutes first time)

In WSL terminal, run:

  buildozer android debug deploy run

What happens:
  ⏳ [00:00-05:00] Downloading Android SDK/NDK (first time)
  ⏳ [05:00-15:00] Compiling Python + Kivy for Android
  ⏳ [15:00-25:00] Compiling NumPy + SciPy
  ⏳ [25:00-30:00] Building APK
  📱 [30:00] Installing on phone
  🚀 [30:30] Launching app!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚡ ALTERNATIVE: Just Build APK (No Auto-Deploy)

If you want to build APK and install manually:

In WSL:
  buildozer android debug

APK will be at:
  bin/chordimporter-0.1.0-arm64-v8a_armeabi-v7a-debug.apk

Then:
  1. Copy APK to phone
  2. Open APK file on phone
  3. Tap "Install"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 SUPER QUICK VERSION

1. Enable USB Debugging on phone
2. Connect phone via USB
3. Open WSL: wsl
4. Run: cd /mnt/f/projetos/chordImporter/android-kivy

🔍 TROUBLESHOOTING

"No devices found"
  → Check USB cable
  → Enable USB debugging
  → Run: adb devices (should show your device)

"Permission denied"
  → Allow USB debugging popup on phone
  → Run: adb kill-server && adb start-server

"Build fails"
  → Run: buildozer android clean
  → Try again: buildozer android debug

"Java not found"
  → Run: sudo apt install openjdk-17-jdk
