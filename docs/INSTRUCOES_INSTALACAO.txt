===============================================
COMO USAR
===============================================

1. EXECUÇÃO SIMPLES:
   - Duplo clique em "MusicalToolsSuite.exe"
   - Aguarde alguns segundos para carregar
   - O dashboard principal será exibido

2. PRIMEIRA EXECUÇÃO:
   - Configure sua API key do Serper (opcional)
   - Acesse "⚙️ Configurações" no dashboard
   - Teste as funcionalidades básicas

===============================================
REQUISITOS DO SISTEMA
===============================================

✅ MÍNIMOS:
- Windows 10 ou superior (64-bit)
- 4 GB RAM
- 500 MB espaço em disco
- Conexão com internet (para buscas)

✅ RECOMENDADOS:
- Windows 11 (64-bit)
- 8 GB RAM
- 1 GB espaço em disco
- Placa de som para recursos de áudio
- Microfone para identificação de acordes

===============================================
SOLUÇÃO DE PROBLEMAS
===============================================

❌ ERRO: "Aplicativo não inicia"
✅ SOLUÇÃO: Execute como administrador

❌ ERRO: "Erro de áudio"
✅ SOLUÇÃO: Verifique drivers de áudio

❌ ERRO: "Busca não funciona"
✅ SOLUÇÃO: Configure API key nas configurações

❌ ERRO: "Visualizações não aparecem"
✅ SOLUÇÃO: Instale Visual C++ Redistributable
